Description
-----------

GUI tool to transform images to cross-stitch patterns.

It can:
* Pixelate the image
* Simplify colors, reducing their count to the given number
* Enumerate the colors, printing color index over each "pixel"
* Add contrast grid overlay

Build
-----
```bash
sbt buildDistr
```
